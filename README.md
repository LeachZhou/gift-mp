# gift-mp

## 小程序规划

### 功能
- 用户第一次进入，有个信息采集，用户填入个人信息，性别、年龄、目前是在工作还是学习，加上现在的时间，判断下一次的节日，根据这些来智能为用户第一次推荐礼物。
- 有个自定义分享朋友圈 http://www.wxapp-union.com/forum.php?mod=viewthread&tid=16552&extra=page%3D3%26filter%3Dtypeid%26typeid%3D6 
- 有个每日清晨 参考网易云音乐
- 首页做成卡片左右滑动消除
